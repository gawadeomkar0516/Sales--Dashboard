
# 📊 Super Data Sales Dashboard

This project presents a comprehensive Sales Dashboard built in Microsoft Power BI to track sales, profit, quantity, regional performance, and monthly trends.

## Overview

The **Super Data Sales Dashboard** is an interactive Business Intelligence project developed in **Microsoft Power BI** to analyze sales performance and provide meaningful business insights. The dashboard enables users to monitor key performance indicators (KPIs), identify sales trends, evaluate regional performance, and support data-driven decision-making through dynamic visualizations.

This project demonstrates practical skills in **Power BI, Power Query, DAX, data modeling, and data visualization**, making it suitable for business reporting and portfolio showcasing.

## Business Insights

The dashboard helps users answer important business questions such as:

- Which region generates the highest sales?
- Which product categories perform the best?
- How do monthly sales and profits change over time?
- Which shipping mode contributes the most to sales?
- What is the overall profit margin?
- Which sub-categories generate the highest revenue?
